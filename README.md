# Tic Tac Toe Game
A simple 2-player Tic Tac Toe game built with **HTML**, **CSS**, and **JavaScript**. Play against a friend locally and test your strategy!

## Features
- 2-player mode
- Win detection
- Draw detection
- Restart / New Game button
- Responsive design for different screen sizes

## 🛠️ How to Play
1. Open index.html in your browser
2. Player 1 starts with "O", Player 2 is "X"
3. Click on a box to make a move
4. First to align 3 marks wins
5. If all boxes are filled and no winner, it’s a draw
6. Click "Restart Game" to play again
